# Algorithms

A collection of scratch implementations for new, classic, and interesting algorithms. 

---

## What is this?
This repository serves as a personal playground for implementing, testing, and exploring various algorithms from scratch. It is designed to track problem-solving progress, optimize data structure layouts, and dive deep into the mechanics of computational logic without relying on heavy external libraries.

---

## Repository Structure
Solutions and implementations are organized by foundational topic folders, then by specific problem or algorithm names:


```

/
├── /
│   ├── solution-0.   ← Initial implementation
│   └── solution-1.   ← Optimized version / Alternative approach

```

### Example:

```

Data Structures & Algorithms/two-integer-sum/solution-0.py
Data Structures & Algorithms/binary-search/solution-0.ts
Advanced/graph-neural-networks/solution-0.py

```

## Areas of Focus
* **Core Data Structures & Algorithms:** Re-implementing essential primitives (trees, graphs, sorting, dynamic programming).
* **Machine Learning & AI Infrastructure:** Scratch-built architectures, layers, and custom token/tensor routing concepts.
* **Hardware & Systems Level Logic:** Algorithms prioritizing memory efficiency, direct bit manipulation, and optimal cache layouts.
